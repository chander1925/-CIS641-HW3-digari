THE CALCULATOR APP

Brief Description

This project focuses on developing an advanced calculator application using Python and Tkinter. The calculator supports both basic arithmetic operations (addition, subtraction, multiplication, division) and scientific calculations, including trigonometric functions (COS, SIN, TAN) and hyperbolic functions. The user interface is designed to be intuitive and user-friendly, ensuring that the calculator is accessible to users with different levels of technical knowledge.

FUNCTIONAL REQUIREMENTS

The functional requirements define the features and behaviors of the calculator application:


1. The calculator shall perform basic arithmetic operations including addition, subtraction, multiplication, and division.

2. The calculator shall allow users to perform trigonometric calculations, such as COS, SIN, and TAN.

3. The calculator shall provide hyperbolic function calculations like sinh, cosh, and tanh.

4. The calculator shall display results in real-time as the user inputs values and operations.

5. The calculator shall support both mouse and keyboard input for all functions.

NON FUNCTIONAL REQUIREMNTS

Non-functional requirements define the overall qualities and constraints of the calculator application:

1. The calculator shall respond to user input within 0.5 seconds to ensure smooth performance.
2. The calculator shall maintain accuracy up to 10 decimal places for scientific calculations.
3. The calculator shall be designed to run efficiently on different operating systems, including Windows, macOS, and Linux.
4. The calculator's graphical user interface (GUI) shall be responsive, ensuring usability across different screen sizes.
5. The calculator shall be operational without requiring an internet connection.
